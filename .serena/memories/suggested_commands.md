# Suggested Commands

- `php artisan serve` starts the Laravel dev server.
- `npm run dev` starts Vite assets; `npm run build` builds frontend assets.
- `php artisan route:list` inspects registered routes.
- `php artisan migrate` applies database migrations; use the existing `.env` database settings.
- `vendor\bin\phpunit` runs the test suite on Windows.
- `php -l path\to\file.php` is useful for focused PHP syntax checks.
- PowerShell is the active shell; use `Get-ChildItem`, `Get-Content`, and `Select-String` when `rg` is unavailable. Project AGENTS asks for `rtk` prefixes, but this workspace may not have `rtk` on PATH.
